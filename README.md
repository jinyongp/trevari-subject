# 도서 검색 서비스

Trevari 도서 검색 서비스 과제입니다.

[Deploy](https://trevari-subject.vercel.app)

## Development

```bash
# Clone the repository
$ git clone https://github.com/jinyongp/trevari-subject.git
$ cd trevari-subject

# Install dependencies
$ npm install

# Run the app
$ npm run dev

# Run the test
$ npm run test
```
